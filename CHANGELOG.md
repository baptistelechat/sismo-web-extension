## [0.1.0] - 2025-06-19

### Added

- First functional version of the web extension
- Display of wind, snow, and seismicity zones for French municipalities (including overseas territories)
- Direct integration into Google search results
- Button to access Géorisques directly with municipality information
- Modern user interface with Tailwind CSS and shadcn/ui
- Support for Chrome and Firefox via manifest V2 and V3
- Automatic extension icon generation system
- Packaging script for extension distribution
- Architecture based on React and TypeScript
- Use of the data.gouv.fr address API for geolocation
- Integration with a public API to retrieve climate data
- Intelligent extraction of postal and municipal codes from Google search
- Optimized build system with Vite
